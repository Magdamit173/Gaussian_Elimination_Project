# Gaussian Elimination Project

##  Overview
**Method:** Gaussian Elimination with **Complete Pivoting** (swaps both rows and columns for maximum stability).

###  Development & Tech Stack
* **Language**: HTML/CSS/JS.
* **Workflow**: The original logic was first written in **Python** due to time constraints.
* **AI Implementation**: Used **Prompt Engineering** via [Gemini](https://gemini.google.com) to port the Python logic to JavaScript. This was done to save time and manage the complexity of the 15x15 matrix structure, though the core logic remains understood and verified.
* **Beautification**: Custom CSS grid layout used to maintain readability for large 15x15 matrix systems.

###  Reference
* **Methodology Guide**: [Gaussian Elimination using Complete Pivoting](https://www.youtube.com/watch?v=HW1AaobnBx0)
